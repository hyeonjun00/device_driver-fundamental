<코드 의도>
Git에서 commit을 작성하기 위해서는 디렉토리에 반드시 파일이 존재해야 한다.
일반적으로는 빈 디렉토리에는 .gitkeep이라는 이름이로 빈 파일을 추가한다.
.gitkeep을 개발자가 일일히 수동으로 만들어 줄 수 있지만 Shell script로 자동화 할 수도 있다.

<코드 내용>
1.파라미터로 대상 디렉토리를 입력받기
2.대상 디렉토리의 하위 디렉토리 중에 빈 디렉토리 찾아내기
3.비어있는 디렉토리에 .gitkeep이라는 빈 파일 생성하기
4.하위 디렉토리 안에 또 다른 하위 디렉토리가 있는 경우도 고려
